# Placement Prediction using Machine Learning

## Overview
This project predicts whether a student will be placed based on CGPA and IQ using different classification algorithms.

## Algorithms Used
- Logistic Regression
- KNN
- Linear SVM
- Kernel SVM
- Decision Tree
- Random Forest

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results

| Model               | Accuracy |
|---------------------|----------|
| Logistic Regression | 85%      |
| KNN                 | 90%      |
| Linear SVM          | 85%      |
| Kernel SVM          | 85%      |
| Decision Tree       | 90%      |
| Random Forest       | 90%      |
